# QPat
### Quantum Photonic Analysis Toolkit

Your path to quantum network characterization.

## Features
- Quantum state tomography and density matrix reconstruction
- Coincidence detection for photon pair analysis
- Visibility and channel characterization
- Realistic fiber channel modeling (BIFROST integration)
- Built on SeQuence quantum network simulator
